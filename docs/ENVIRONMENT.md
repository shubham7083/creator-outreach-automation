# Environment Variables

Use `.env.example` as the source of truth for required configuration.

## Core

- `APP_NAME`: UI and logging display name.
- `APP_ENV`: `development`, `test`, `staging`, or `production`.
- `LOG_LEVEL`: Python logging level.
- `DATABASE_URL`: SQLite URL.
- `CACHE_DIR`: cache folder.
- `OUTPUT_DIR`: generated output folder.
- `PROMPTS_DIR`: prompt template folder.
